Architecture Visual

Simple view of how user input moves through your cloud stack and returns a generated pet recipe.

Browser FormUser enters pet details
→
API GatewayPOST /recipes
→
AWS LambdaCreates prompt + handles response
→
Google Gemini APIGenerates recipe text
→
Response to BrowserRecipe shown in UI
Security: API key is stored in AWS Secrets Manager, not hardcoded in frontend files.
Hosting: Frontend can be served from S3 + CloudFront while API stays behind API Gateway.
Scalability: Lambda scales automatically based on traffic volume.
