How the Pet Recipe App Works

This app helps pet owners generate recipe ideas quickly. You enter pet details, the app sends a request to your cloud API, and the backend returns a generated recipe.

Challenge

Give non-technical users a simple way to generate pet-friendly recipe ideas without dealing with cloud setup details.

Frontend

A simple form collects pet type, ingredients, and allergies, then sends one request to the API endpoint.

Backend

API Gateway routes the request to Lambda. Lambda creates the prompt, calls Gemini, and returns the recipe response.

Security

The Gemini key is stored in AWS Secrets Manager and read at runtime. It is not hardcoded in frontend or backend files.
